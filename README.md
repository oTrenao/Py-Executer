# Py-Executer
Desenvolvedor focado em automação com Python, modding e ferramentas de otimização. Criador do Sthevan P.K. (oTrenao), uma interface completa para gestão de ambientes e compilação de executáveis.

Sobre o Projeto
O Py Executer Ultimate é uma ferramenta baseada em HTA (HTML Application) projetada para facilitar o fluxo de trabalho de desenvolvedores Python no Windows. Ele centraliza a execução de scripts, gerenciamento de ambientes virtuais (VENV) e a compilação de executáveis otimizados através de uma interface moderna e intuitiva.

✨ Principais Funcionalidades

Gestão de Scripts: Lista e executa arquivos .py diretamente da pasta do projeto com um clique.


Controle de Ambientes (VENV): Criação e remoção automática de ambientes virtuais para isolar dependências.

Build Avançado de Executáveis:

Suporte nativo para PyInstaller e Nuitka (compilação para C++).

Integração com UPX para compressão extrema do tamanho final do .exe.

Obfuscação de código via PyArmor para proteção de propriedade intelectual.


Smart Scan: Identifica automaticamente dependências necessárias e módulos que podem ser excluídos para otimizar o build.


Instalação Automática: Ferramenta de "AutoInstall" que detecta e instala bibliotecas faltantes via pip.

🛠️ Tecnologias Utilizadas

Front-end: HTML5/CSS3 com Design System Dark Modern.


Scripting: JScript (ActiveX) para interação direta com o sistema de arquivos do Windows e Shell.


Back-end: Python (gestão de venv, execução de processos).

🎮 Como Usar
Certifique-se de ter o Python instalado e no PATH do sistema.

Coloque o arquivo Py Executer Ultimate V2.hta na pasta raiz dos seus projetos Python.

Execute o arquivo e utilize a barra lateral para selecionar seu script.

Configure o ambiente (VENV) na aba de configurações antes de rodar ou compilar.
